# Macy Xiang Portfolio

Personal portfolio site for Macy Xiang, built with React and Vite.

## Features

- Bilingual English/Chinese content
- Project, experience, education, and skills sections
- Resume, GitHub, LinkedIn, and email links
- Static build suitable for GitHub Pages

## Tech Stack

- React
- Vite
- CSS

## Local Development

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Run lint:

```bash
npm run lint
```
